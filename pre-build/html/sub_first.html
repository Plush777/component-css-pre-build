component css 소개

component css 사이트는 웹 퍼블리싱 작업에 필요한 여러가지 요소들의 스타일들을 정리하여 제공하는 사이트입니다.

물론 모든 스타일을 다 제공할 수는 없겠지만 어느정도 기본 틀이되는 css 컴포넌트들을 제공하여 원하는 색상이나 모양을 바꿔서

그대로 쓸 수 있는 컴포넌트들입니다.

목적

웹사이트에 자주 사용되는 css 스타일들을 기록해서 바로바로 코드를 긁어서 사용할 수 있도록 하는 것이 주 목적입니다.
